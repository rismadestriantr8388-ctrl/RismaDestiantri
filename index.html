Tentang Saya

Merangkai Kode, Menciptakan Karya

Saya adalah mahasiswa Informatika yang sedang mempelajari dunia teknologi dan pengembangan web. Saat ini saya masih berada di tahap pemula, tetapi saya memiliki semangat yang tinggi untuk terus belajar dan mengembangkan kemampuan. Saya tertarik mempelajari dasar-dasar pembuatan website, mulai dari HTML, CSS, JavaScript, hingga berbagai teknologi web lainnya.

Menurut saya, belajar web bukan hanya tentang membuat tampilan website yang menarik, tetapi juga memahami bagaimana sebuah website dapat berjalan dengan baik dan memberikan manfaat bagi penggunanya. Karena itu, saya terus berusaha meningkatkan kemampuan melalui perkuliahan, latihan mandiri, serta mencoba membuat proyek-proyek sederhana sebagai sarana belajar.

Saya percaya bahwa proses belajar membutuhkan waktu, konsistensi, dan kemauan untuk terus mencoba. Sebagai mahasiswa Informatika, saya ingin terus berkembang agar memiliki keterampilan yang dapat diterapkan di dunia kerja maupun dalam berbagai proyek di masa depan.
